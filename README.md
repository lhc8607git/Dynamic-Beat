## 👋 소개
“Music Game”은 자바 플랫폼에서 사용할 수 있는 GUI로 AWT와 Swing 활용해서 리듬
게임을 개발하였습니다. 전반적인 GUI구성과 상단에서 내려오는 노드를 하단부에 판단하는
바에 접근 시, 각각의 블록에 해당 키보드를 누르면, 지정한 범위 안의 Miss, Great, Good 
등이 표시되도록 개발하였습니다.
